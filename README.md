# 🍪 CRK Map Art Planner

[![Открыть сайт](https://img.shields.io/badge/🚀_Открыть_сайт-2ea44f?style=for-the-badge)](https://pirat3571.github.io/crkmapart/)

Инструмент для помощи в построении мап-артов на лужайке в **Cookie Run: Kingdom**.  
Можно делать пиксель-арты прямо в браузере, используя настоящие игровые тайлы.

## ✨ Возможности

- **Изометрическая сетка** 64×60, точно повторяющая форму лужайки в CRK (области, доступные для строительства, взяты из маски игры).
- **Наложение изображения** – загрузите картинку, подгоните её положение и размер, а затем автоматически превратите в расстановку тайлов.
- **Умное сопоставление цветов** – алгоритм подбирает наиболее подходящий тайл по усреднённому цвету каждой ячейки.
- **Ручной режим** – кисть, ластик и заливка, чтобы дорабатывать арт или рисовать с нуля.
- **Экспорт в PNG** – сохраните текущее состояние карты в виде изображения.

## 🔩 Локальный запуск

1. Клонируйте репозиторий или просто скачайте файлы.
   ```bash
   git clone https://github.com/pirat3571/crkmapart.git
   ```
2. Откройте `index.html` двойным кликом, должен открыться браузер.
3. Загрузите **исходное изображение**, которое хотите перенести на лужайку, подгоните его положение и масштаб.
4. Загрузите **картинки плиток** (.webp‑файлы в папке `\tiles`), выделить все - Ctrl+A.  
5. Нажмите **«Применить изображение»**, и ячейки на карте заполнятся ближайшими по цвету плитками.

   Опционально:
- Доработайте арт вручную: выберите плитку в палитре, рисуйте кистью, стирайте, заливайте области.
- Когда всё готово, экспортируйте результат как PNG.

## 📐 Изометрическая проекция

Карта повёрнута на 45° и сжата по вертикали в два раза (стандартный dimetric‑проект под углом ~26.565°).  
Координаты в сетке: ось столбцов идёт влево‑вверх, ось строк – вправо‑вверх.  
Пустые (чёрные) клетки соответствуют области за пределами лужайки – они не редактируются.

## ♿ Управление

| Действие               | Как выполнить                                      |
|------------------------|----------------------------------------------------|
| Рисовать               | Выбрать тайл и щёлкните/проведите по ячейке        |
| Стереть                | Инструмент «Ластик»                                |
| Залить свободные       | Инструмент «Залить»                                |
| Панорама               | Пробел + перетаскивание                            |
| Зум                    | Колёсико мыши                                      |
