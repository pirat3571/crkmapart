# 🍪 CRK Map Art Planner

[![Открыть сайт](https://img.shields.io/badge/🚀_Открыть_сайт-2ea44f?style=for-the-badge)](https://pirat3571.github.io/crkmapart/)

Инструмент для помощи в построении мап-артов на лужайке в **Cookie Run: Kingdom**.  
Можно делать пиксель-арты прямо в браузере, используя настоящие игровые тайлы.

## ✨ Возможности

- **Изометрическая сетка** 64×60(256×240), точно повторяющая форму королевства в CRK (маска области, доступной для строительства, взята из игры).
- **Наложение изображения** – загрузите картинку, подгоните её положение и размер, а затем превратите в расстановку цветных тайлов и декораций.
- **Умное сопоставление цветов** – алгоритм подбирает наиболее подходящий тайл по усреднённому цвету каждой ячейки.
- **Ручной режим** – кисть, ластик и заливка, чтобы дорабатывать арт или рисовать с нуля.
- **Экспорт в PNG/XML** – сохраните текущее состояние карты в виде изображения или файла для постройки позже.

## 🔩 Локальный запуск

1. Клонируйте репозиторий или просто скачайте файлы.
   ```bash
   git clone https://github.com/pirat3571/crkmapart.git
   ```
2. Откройте `index.html` двойным кликом, должен открыться браузер.
3. Загрузите **исходное изображение**, которое хотите перенести на карту, подгоните положение и масштаб.
4. Загрузите **картинки плиток** (.webp‑файлы в папке `\tiles`).
5. Загрузите **картинки декорвций** (.png‑файлы в папке `\decorations`).
6. Нажмите **«Применить изображение»**, и ячейки на карте заполнятся ближайшими по цвету плитками.

   Опционально:
- Доработайте арт вручную: выберите плитку в палитре, рисуйте кистью, стирайте, заливайте области.
- Когда всё готово, экспортируйте результат как PNG.

## 📐 Изометрическая проекция

Карта повёрнута на 45° и сжата по вертикали в 2 раза (стандартный изометрический вид).  
Пустые (чёрные) клетки соответствуют области за пределами карты – в игре они заблокированы и не редактируются.

## ♿ Управление

| Действие               | Как выполнить                                      |
|------------------------|----------------------------------------------------|
| Рисовать               | Выбрать тайл и щёлкните/проведите по ячейке        |
| Стереть                | Инструмент «Ластик» или ПКМ                        |
| Залить свободные       | Инструмент «Залить»                                |
| Передвигать камеру     | Пробел + перетаскивание                            |
| Зум                    | Колесико мыши                                      |
